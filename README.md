

# 📝 RELOJ IB

**RELOJ IB** es una herramienta web de alta precisión diseñada para la gestión del tiempo durante exámenes o pruebas académicas. Combina un visor de documentos dinámico con un sistema de cronometraje inteligente que contempla periodos de pre-lectura y ajustes manuales en tiempo real.

---

## ✨ Características Principales

* **🕒 Reloj Inteligente**: Seguimiento en tiempo real con precisión de segundos.
* **📖 Periodo de Pre-lectura**: Sistema integrado de 5 minutos de lectura previa antes del inicio formal del examen.
* **⚙️ Override Manual (Ajuste Quirúrgico)**: Mediante el **clic derecho**, puedes forzar la hora exacta de inicio del examen. El sistema recalculará automáticamente:
    * La cuenta atrás de la pre-lectura.
    * La barra de progreso actual.
    * La hora exacta de finalización.
* **📄 Visor de Archivos Integrado**: Carga PDFs o imágenes directamente en la herramienta.
* **📏 Panel Compacto Resizable**: Al hacer scroll para leer el documento, el reloj se fija en la parte inferior y permite **ajustar su altura verticalmente** para no estorbar la lectura.
* **🌓 Modo Dual**: Soporte para Modo Claro (por defecto) y Modo Oscuro con iconos SVG vectoriales.
* **📍 Marcadores Críticos**: Visualización de marcas visuales para los últimos 30 y 5 minutos del examen.

---

## 🚀 Cómo Usar

1.  **Configuración**: Selecciona la duración del examen en el menú desplegable.
2.  **Carga**: Sube el archivo del examen (PDF o Imagen) mediante el botón "Archivo".
3.  **Inicio Estándar**: Pulsa "Iniciar Lectura" para comenzar los 5 minutos de lectura seguidos del examen.
4.  **Ajuste Manual (Override)**: 
    * Haz **clic derecho** en cualquier parte de la pantalla.
    * Ingresa la hora exacta en la que debe iniciar el examen.
    * Pulsa "Actualizar hora" y el sistema sincronizará todo el progreso al instante.
5.  **Lectura**: Haz scroll hacia abajo para leer el documento; el reloj se mantendrá visible y podrás estirarlo o encogerlo desde la esquina inferior derecha.

---

## 🛠️ Tecnologías Utilizadas

* **HTML5**: Estructura semántica.
* **CSS3**: Diseño responsivo, variables CSS, Flexbox y animaciones de transición.
* **JavaScript (Vanilla)**: Lógica de cronometraje, gestión de estados y manipulación dinámica del DOM.
* **SVG**: Iconografía vectorial para una carga ligera y escalable.

---

## 📋 Requisitos

No requiere instalación ni dependencias externas. Al ser un archivo único (Single File Application), solo necesitas un navegador moderno (Chrome, Firefox, Edge o Safari).

---

> **Tip**: El modo compacto ajustable es ideal para pantallas de tablets o laptops donde el espacio vertical es valioso.
